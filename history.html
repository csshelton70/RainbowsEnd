Rainbow's End Version History

The way I increment version numbers is:

  • Bug fixes and small adjustments get a .01 increment.
  • Changes that make a small difference in how the game is played get a .1 increment.
  • Changes that make a big difference in how the game is played, go to the next integer version number.
